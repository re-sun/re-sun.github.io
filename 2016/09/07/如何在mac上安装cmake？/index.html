如何在mac上安装cmake？

By Sun

2016-09-07 更新日期:2016-09-07

由于在mac上安装opencv需要使用到cmake，故今日尝试了一下，很简单。

1、前往cmake官网，http://www.cmake.org/download/，下载cmake安装程序

Unix/Linux Source (has \n line feeds)    cmake-3.6.1.tar.gz

2、解压下载的安装包后，进入到cmake目录，执行以下命令

sudo ./bootstrap && sudo make && sudo make install

输入你的电脑账户密码后开始安装
